# Soul Nest 記帳系統

這是一個可以直接放到 GitHub Pages / Vercel 使用的靜態網頁版記帳系統。

## 功能

- 儀表板：查看本期總金額、本期淨利、全部總金額、全部淨利
- 年份 / 月份切換：可看單月，也可看全年
- 新增、編輯、刪除記帳資料
- 記錄客人 / 來源，例如暱稱、IG、LINE
- 本期收入、成本、淨利、已收款、未收款自動統計
- 分類收入與分類淨利統計
- 年度月份趨勢
- 年度月報表
- 常用品項設定，新增記帳時自動帶入單價與成本
- 欄位設定：可顯示 / 隱藏欄位、修改欄位名稱、調整欄位順序
- CSV 匯出會依照目前顯示欄位輸出
- JSON 備份與匯入
- 手機與電腦都可使用

## 使用方式

1. 把這個資料夾上傳到 GitHub Repository。
2. 使用 GitHub Pages 或 Vercel 部署。
3. 開啟網站後即可開始記帳。

## 欄位設定

到「系統設定」頁面，可以調整記帳明細欄位：

- 勾選是否顯示欄位
- 修改欄位名稱
- 使用上移 / 下移調整順序
- 按「恢復預設欄位」可還原

## 重要提醒

目前資料會存在「目前瀏覽器」的 localStorage。

也就是說：

- 同一台電腦、同一個瀏覽器會看得到資料。
- 換手機或換電腦不會自動同步。
- 換裝置前請先按「備份 JSON」。

如果你要多人共同使用、手機電腦同步、登入帳號，可以再接 Supabase 資料庫。

## 視覺風格

這個版本已調整為亮色系，使用米白、綠色與金色，讓系統看起來更接近 Soul Nest Logo 的品牌感。


## 新增自訂欄位

到「系統設定」→「欄位設定」可以新增自訂欄位，例如：

- 老師名稱
- 老師分潤
- 平台來源
- IG 帳號
- LINE 名稱
- 儀式狀態

新增後，記帳明細表會出現該欄位，新增 / 編輯記帳時也會出現可填寫的欄位。


## 自動分類與分組

記帳明細現在預設會「依分類分組」，同一類型會自動放在一起，並顯示每一組的小計：

- 筆數
- 總金額
- 成本
- 淨利
- 未收款

也可以在記帳明細上方切換：

- 依分類分組
- 依日期排序
- 依客人 / 來源分組
- 依付款狀態分組

新增記帳時，輸入品項也會依關鍵字自動判斷分類，例如「塔羅、儀式、讀心、水晶、代理、成本」等。


## 分潤設定

到「系統設定」→「品項設定」可以設定每個服務：

- 客人收費 / 件
- 給老師 / 成本 / 件
- 系統會自動算「我們抽 / 件」

例如：

- 感情儀式：客人收費 3000，給老師 1500，系統會自動算我們抽 1500。
- 新增記帳時選到該品項，會自動帶入客人收費與給老師金額。
- 數量大於 1 時，給老師金額會自動乘上數量。
